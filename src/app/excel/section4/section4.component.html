Skapa din första Angular-app med Angular CLI

Steg 1: Installera Node.js och NPM

För att komma igång med Angular behöver du installera Node.js och NPM. Besök den officiella webbplatsen för att hämta den senaste versionen. Kontrollera att du har en tillräckligt ny version av Node.js genom att köra kommandot node --version.

Steg 2: Installera Angular CLI

Angular CLI är ett kraftfullt verktyg för att snabbt starta och konfigurera Angular-projekt. Det underlättar processen genom att konfigurera Webpack, tester och paketering automatiskt. Installera Angular CLI genom att köra följande kommando:

npm install -g angular/cli

Steg 3: Skapa din första komponent

I Angular 15 introducerades funktionen "standalone components", vilket förenklar strukturen av Angular-projekt. Vi kommer att använda denna funktion i vår tutorial.

Nu, skapa din första komponent med Angular CLI:

ng generate component my-component

Steg 4: Struktur på din Angular-app

Din Angular-apps rotmapp kommer nu att innehålla flera konfigurationsfiler, inklusive package.json, tsconfig.json, angular.json, och .eslintrc.json. Dessa filer används för att konfigurera och hantera olika aspekter av din app.

Summering (på svenska)

  • Installera Node.js och NPM.
  • Installera Angular CLI.
  • Skapa ett nytt Angular-projekt med Angular CLI.
  • Skapa din första komponent.
  • Utforska och förstå projektstrukturen och konfigurationsfiler.

Lycka till med din Angular-resa! 🚀
